# Deep Learning Projects

A collection of deep learning and agentic AI projects completed as part of my Deep Learning coursework at the University of Mississippi.

The projects focus on implementing neural network architectures from scratch, experimenting with modern attention mechanisms, and building autonomous AI workflows.

## Projects

### Project 2 — RNN & GRU Language Modeling

Implemented recurrent neural networks and gated recurrent units from scratch using PyTorch for character-level language modeling on the Time Machine dataset.

Topics:
- Vanilla RNN
- GRU
- Hyperparameter tuning
- Gradient clipping
- Text generation

---

### Project 3 — Attention Mechanisms

Implemented attention mechanisms used in transformer architectures.

Topics:
- Basic Attention
- Multi-Head Attention
- PyTorch implementation

---

### Project 5 — LaTeX Optimizer Agent

Designed an autonomous AI workflow that optimizes large LaTeX projects while preserving document quality.

Topics:
- Agentic AI workflow
- Image compression
- Automated LaTeX compilation
- Project size optimization
